If $p_s = 1-\frac {1}{n^2}$ then $p_e = \frac {1}{n^2}$, since $\log \frac {m}{n}$ are done, the global probability of failure $P_f$ is the sum of each $p_f$, such that $$P_f = \frac {1}{n^2} \times \log \frac {m}{n}.$$ Since $m$ is the number of edges of the input graph, $$m \leq \binom {n}{2}\in O(n^2),$$ thus $$\log \frac mn \in O\big (\log \frac {n^2}{n}\big ) = O(\log n)\text { ,}$$ hence $$p_e = O\big (\frac {\log n}{n^2}\big )\text { ,}$$ the result follows.\end{proof}
